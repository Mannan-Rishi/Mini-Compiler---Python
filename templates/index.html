Academic Architecture Features

Function Declarations

Supports int, float, and void functions with local scoping, formal parameters, and custom return expression validation.

Semantic Integrity

Detects scope violations, duplicate variable declarations, undeclared identifier usage, and float-to-int type mismatches.

IR Optimizations

Applies Constant Folding and Propagation, alongside Dead Code Elimination on intermediate Three-Address Code (TAC).

Control Flow CFG

Identifies basic block leaders, splits block instructions, and renders directed graphs mapping branch and fallthrough paths.
